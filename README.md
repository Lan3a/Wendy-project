
## Clone / Download
Clone / Download the repo, 
rename folder to "scripts" and put it inside a master folder,
and create other folders as below (if need to put data in them)

e.g.
```
└── Wendy's project
    └── scripts
    └── raw
    └── preprocess
    └── analysis
```

- raw        - put .edf raw eeg files
- preprocess - eeg .set generated during preprocessing
- analysis   - eeg features and analysis

## How to use
identify a verion for runPreprocess / runAnalysis, 
navigate to within scripts folder, then execute codes section-by-section
